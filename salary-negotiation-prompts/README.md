## Research & Preparation

```
I am a [job title] with [X] years of experience in [city/country]. What is the market salary range for my role in [year]? What sources should I check and what factors affect the range?
```

```
I have a job offer for [role] at [company type] paying [amount]. Is this competitive for [city] in [year]? What should I counter with?
```

```
Help me build a case for a salary negotiation. My role is [title], I have been at the company [X] years, I earn [current salary], and the market rate is [market rate]. Write a structured argument.
```

```
What are 5 things I should research before negotiating my salary for a [role] at a [company size] company?
```

```
I am switching from [current role, current salary] to [new role]. What salary should I target and how much increase is reasonable to ask for?
```

## Initial Offer Negotiation

```
I received a job offer of [amount] for a [role] at [Company]. I was hoping for [target amount]. Write a professional email to negotiate higher without sounding ungrateful or aggressive.
```

```
Write a script for a 5-minute phone call where I counter a job offer of [amount] and ask for [target amount]. I have [X] years of experience and a competing offer of [amount]. Keep it confident but collaborative.
```

```
The recruiter asked me what salary I am looking for before giving me a number. Write 3 different responses I can use to deflect and ask them to go first.
```

```
A company gave me an offer but said the salary is fixed and non-negotiable. Write a script to negotiate other parts of the compensation package: signing bonus, extra PTO, remote work, earlier review date.
```

```
Write a counter-offer email. Offer received: [amount]. My target: [amount]. Reason: I have [competing offer / market data / X years experience]. Keep it under 150 words and professional.
```

## Asking for a Raise

```
Write a script for asking my manager for a salary increase. I have been in this role for [X] years, I earn [current amount], market rate is [market rate], and my recent achievements include [achievement 1] and [achievement 2].
```

```
I want to ask for a [X]% raise but my company had a tough year. Write a persuasive script that acknowledges the situation but still makes a strong case based on my contributions.
```

```
Help me prepare for a performance review where I plan to ask for a raise. My role: [title]. Key wins this year: [list 3]. What should I say and how should I frame the ask?
```

```
Write an email to my manager requesting a meeting to discuss my compensation. Keep it brief, professional, and do not reveal the exact ask in the email itself.
```

```
My manager said they will think about the raise request and get back to me. It has been 3 weeks. Write a polite follow-up email.
```

## Handling Pushback

```
My manager said the budget is frozen and they cannot give me a raise right now. Write a professional response that keeps the conversation open and sets up a timeline for revisiting it.
```

```
The hiring manager said my counter-offer is too high. Write a response that holds my position while showing flexibility on non-cash benefits.
```

```
My employer said they are already paying me at the top of the band. Write a professional response that challenges this politely and asks for transparency on the band range.
```

```
I was told I need to wait until my next performance review in 6 months for a raise. Write a response that gets a written commitment and an agreed target.
```

```
The recruiter came back with only a small increase from their original offer. Write a response that pushes one more time without risking the offer.
```

## Total Compensation

```
A company offered me [base salary] but I currently earn [total comp including bonus and equity]. Write a script to negotiate total compensation, not just base salary.
```

```
Explain the difference between base salary, total cash compensation, and total compensation including equity. Write a script where I use this knowledge to negotiate a better overall package.
```

```
I want to negotiate a signing bonus to offset the unvested equity I am leaving behind at my current job. Write an email explaining this and asking for a specific amount.
```

```
Write a script to negotiate for 100% remote work as part of my offer. Frame it in terms of productivity and what I bring to the role.
```

```
I want to negotiate an earlier performance review date (3 months instead of 12). Write a script that ties this to clear performance targets.
```

## Specific Scenarios

```
I am a woman negotiating my salary in a male-dominated industry. Write a confident, assertive negotiation script that does not apologize for the ask.
```

```
I am a recent graduate with no prior salary negotiation experience. Write a simple, friendly but firm script to negotiate my first offer up by [amount].
```

```
I have two competing job offers. Write a script to use one offer to improve the other without damaging either relationship.
```

```
I am a freelancer transitioning to full-time employment. My freelance rate was [amount/hour]. Write a script to convert this to a competitive full-time salary ask.
```

```
I want to negotiate a remote work stipend, gym membership, and learning budget as part of my package. Write a single email that asks for all three professionally.
```
