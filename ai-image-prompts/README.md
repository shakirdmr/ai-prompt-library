# AI Image Prompts — 50 Free Copy-Paste Examples

## What is this?

This is a collection of 50 ready-made prompts for generating images with AI tools like **Midjourney**, **DALL-E 3** (inside ChatGPT), **Adobe Firefly**, and **Gemini**.

If you have ever typed something like *"a woman at a cafe"* and got a flat, generic result — this is why. AI image tools need detail: lighting, camera angle, mood, style, and context. The prompts below are already written with all of that built in.

## How to use this

1. Find a prompt in the category that matches what you want to create
2. Copy the full prompt from inside the code block
3. Paste it into your AI image tool (ChatGPT with DALL-E, Midjourney, Firefly, etc.)
4. Replace any part in `[brackets]` with your own details
5. For even better results, enhance the prompt first at [PromptIt](https://promptitin.com/blog/prompts-for-ai-image-generator)

> **Tip for beginners:** If you are using ChatGPT to generate images, just paste the prompt into the chat and it will automatically use DALL-E to create the image. No extra steps needed.
