## Writing Your Resume from Scratch

```
Write a professional resume for a [job title] with [X] years of experience in [industry]. Include a summary, work experience at [Company A] and [Company B], skills section, and education. Use strong action verbs and quantify achievements where possible.
```

```
I am a [job title] applying for a [target role] at a [company type]. Write my resume summary in 3-4 sentences that highlights my [top skill], [second skill], and [achievement]. Make it ATS-friendly.
```

```
Write 5 bullet points for my role as [Job Title] at [Company]. My responsibilities included [list tasks]. Rewrite these using the STAR method (Situation, Task, Action, Result) with specific metrics.
```

```
I have [X] years of experience in [field] but I am switching careers to [new field]. Write a resume summary that bridges my existing skills to the new role without highlighting the career gap.
```

```
Create a one-page resume for a recent graduate with a degree in [subject] from [University]. Include 2 internships at [Company A] and [Company B], a GPA of [X], and skills in [skill1], [skill2], [skill3].
```

## Improving Existing Bullet Points

```
Rewrite these resume bullet points to be more impactful with quantified results: [paste your bullets here]. Use strong action verbs and include percentages, dollar amounts, or timeframes where relevant.
```

```
My resume bullet says: "[paste weak bullet]". Rewrite it to sound more impressive while staying truthful. Add a measurable result if one can be reasonably inferred.
```

```
Turn these job duties into achievement-focused resume bullets. Make each one start with a different strong action verb: [paste duties list].
```

```
I managed a team and improved our process but I do not have exact numbers. Write 3 resume bullets that sound credible without fabricating specific metrics.
```

```
Rewrite my entire experience section for a [job title] role. Here is what I currently have: [paste experience section]. Make it tighter, more results-focused, and ATS-optimized.
```

## Tailoring for a Specific Job

```
Here is a job description: [paste JD]. Here is my current resume: [paste resume]. Rewrite my resume summary and top 3 bullet points to align with the keywords and requirements in this job description.
```

```
Extract the top 10 keywords from this job description that I should include in my resume: [paste JD].
```

```
I am applying to [Company Name]. Research their values and culture and rewrite my resume summary to reflect what they look for in candidates.
```

```
My resume is for a [role] in [industry]. Rewrite it to pass ATS screening for this job description: [paste JD]. Highlight keyword matches and suggest what to add.
```

```
Compare my resume to this job description and tell me: (1) what matches well, (2) what is missing, (3) what to cut. Resume: [paste]. JD: [paste].
```

## Skills Section

```
I have experience in [list tools and skills]. Organize these into a clean, professional skills section for a [job title] resume. Group them into categories like Technical Skills, Soft Skills, and Tools.
```

```
What are the top 10 skills I should list on a resume for a [job title] role at a [company type] in [year]? Focus on what recruiters and ATS systems look for.
```

```
I am a [job title]. Write a skills section that balances hard skills like [skill1, skill2] with soft skills like [skill3, skill4] — formatted for a modern resume.
```

## Cover Letter Companion

```
Write a cover letter for a [job title] position at [Company]. I have [X] years of experience in [field] and my top achievements include [achievement 1] and [achievement 2]. Keep it to 3 paragraphs, confident but not arrogant.
```

```
I am applying to [Company] for a [role]. Write a cover letter opening paragraph that immediately grabs attention and shows I understand their product or mission.
```

```
Write a cover letter for someone switching from [old career] to [new career]. Address the career change directly and frame past experience as an advantage.
```

## LinkedIn Optimization

```
Rewrite my LinkedIn headline for a [job title] who specializes in [skill] and is open to [type of opportunities]. Make it keyword-rich and compelling. Current headline: [paste].
```

```
Write a LinkedIn About section for a [job title] with [X] years of experience. Include: what I do, who I help, key achievements, and a call to action. Keep it under 300 words and conversational.
```

```
Turn my resume bullet points into LinkedIn experience descriptions. Make them slightly more conversational and readable: [paste bullets].
```

## Handling Gaps & Tricky Situations

```
I have a 2-year gap in my resume from [year] to [year]. I was [briefly explain: traveling / caregiving / freelancing / studying]. Write 2 ways to address this in my resume and cover letter honestly but positively.
```

```
I was laid off from my last role. How should I explain this in my resume, cover letter, and LinkedIn profile without it raising red flags? Write example language for each.
```

```
I have done a lot of freelance work but no full-time employment for 3 years. How should I structure this on my resume so it looks strong and professional?
```

```
I worked at the same company for 10 years and got promoted twice. How do I show multiple roles at the same company on my resume to highlight growth?
```

## Interview Prep from Resume

```
Based on my resume below, generate 10 interview questions a hiring manager is likely to ask me and write strong answers using the STAR method: [paste resume].
```

```
What weaknesses on my resume might a recruiter notice? How should I address them proactively in an interview? Resume: [paste].
```

```
Write a 60-second elevator pitch I can use in an interview based on my resume: [paste resume].
```
